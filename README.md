# Smart Gas Leakage Detection and Alert System

## Overview

This project implements an IoT-based smart gas monitoring system using ESP32. It continuously monitors gas concentration, temperature, and humidity and uploads the data to ThingSpeak for cloud monitoring.

## Features

- Real-time gas monitoring
- Temperature & humidity monitoring
- OLED live display
- RGB hazard indication
- Buzzer alarm
- Cloud dashboard using ThingSpeak
- ESP32-based implementation
- Wokwi simulation

## Hardware

- ESP32
- MQ2 Gas Sensor
- DHT22
- SSD1306 OLED
- RGB LED
- Active Buzzer

## Software

- Arduino IDE
- Wokwi
- ThingSpeak
- GitHub

## Results

- Gas concentration displayed locally
- Data uploaded every 15 seconds
- Real-time cloud visualization

## Future Improvements
